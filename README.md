# Portfolio

A dark neon personal portfolio built with React + Vite.
Showcases my experience in content writing, social media management,
and my journey into tech.

## 🔗 Live Demo
https://vyraadev.github.io/portfolio/

## ✦ Sections
- Home — Hero with typewriter animation
- About — Bio, quick facts, terminal card
- Skills — Animated skill bars
- Experience — Tab-based job timeline
- Contact — Social links + copy email

## 🛠 Built With
- React 18
- Vite
- Pure CSS
- Deployed on GitHub Pages
